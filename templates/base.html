Spine osteopororsis detection

It leverages deep learning models to automatically analyze and detect bone health issues with high accuracy

📘 About the Project

This project aims to detect osteoporosis using deep learning models applied to spine X-ray images. It enhances early diagnosis by automating image analysis, improving accuracy and reducing manual effort.

🧠 How It Works

  • Image Upload: Users upload spine X-ray images through the web interface for analysis.
  • Preprocessing: The system resizes and normalizes the X-ray images to prepare them for model input.
  • Deep Learning Prediction: A trained CNN model analyzes the image to detect signs of osteoporosis.
  • Result Generation: The system displays the predicted label and confidence level to the user.

🦴 Features

  • Automated detection of osteoporosis from spine X-ray images using deep learning.
  • Accurate prediction with confidence scores for medical reliability.
  • User-friendly interface for uploading and viewing diagnostic results.
  • Admin dashboard for managing user data and monitoring system performance.
