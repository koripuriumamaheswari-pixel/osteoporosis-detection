📊 Overview

This admin dashboard allows monitoring and managing all aspects of the osteoporosis detection system using spine X-ray images. The system leverages deep learning to detect osteoporosis effectively from radiographic data.

🛠 Features

  • Admin approval for user registration and access control.
  • Dataset upload and review for spine X-ray images.
  • Model training management using convolutional neural networks (CNN).
  • Prediction and analysis of osteoporosis from X-ray input.
  • Real-time user and system monitoring with explainable AI outputs.

👨‍⚕️ Purpose

The goal of this system is to provide a robust, automated solution to assist radiologists and clinicians in detecting osteoporosis, reducing manual effort and increasing diagnostic accuracy.
